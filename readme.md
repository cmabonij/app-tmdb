### Dep versions:
- React Native v0.71.13
- React v18.2.0

### Execute project
- `npm run android`
- `npm run android:linux`
- `npm run ios`

### Build
`npm run build`

### Install dependencies to execute
1. download [node](https://nodejs.org/en/download/)
2. npm install
3. cd /ios && pod install
4. npm run ios / android / android:linux
